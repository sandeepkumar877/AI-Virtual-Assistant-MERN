# 🤖 AI Virtual Assistant Using MERN Stack

An AI-powered Virtual Assistant built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). The application provides secure user authentication, intelligent AI-powered conversations, and a responsive user interface.

---

## 🚀 Features

- 🔐 User Authentication (JWT)
- 🤖 AI Chat Assistant
- 💬 Real-time AI Responses
- 🗂️ Chat History
- 🔒 Secure Login & Signup
- 📱 Responsive Design
- 🌐 REST API Integration
- ⚡ Fast and User-Friendly Interface

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication
- JWT (JSON Web Token)
- bcrypt

### AI Integration
- OpenAI API / Gemini API

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/sandeepkumar877/AI-Virtual-Assistant-MERN.git
```

### Go to Project Folder

```bash
cd AI-Virtual-Assistant-MERN
```

### Install Backend Dependencies

```bash
cd server
npm install
```

### Install Frontend Dependencies

```bash
cd ../client
npm install
```

---

## ▶️ Run the Project

### Start Backend

```bash
cd server
npm run dev
```

### Start Frontend

```bash
cd client
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file inside the `server` folder.

```env
PORT=5000

MONGODB_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

OPENAI_API_KEY=your_openai_api_key
```
